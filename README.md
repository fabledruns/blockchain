# Go Blockchain

<p align="center">
  <img src="assets/banner.png" alt="Go Blockchain Banner" width="900">
</p>

A simple blockchain implementation written in Go for learning how blockchains work. It includes block creation, SHA-256 hashing, block chaining, and Proof of Work (PoW) mining. The project is built step by step to understand the core concepts behind blockchain technology.

More features (and probably a few bugs) will be added as development continues.
